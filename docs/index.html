Key Features

FeatureDescription
User AuthenticationRegistration, login, 3-step password reset via PIN or security question
Budget PlannerSet monthly income and fixed/variable expense budgets
Receipt ScannerLog actual spending by category; compare budget vs. actual
Bill TrackerTrack bills with due dates, reminders, recurrence, and payment history
DashboardVisual summary of income, expenses, savings rate, and spending pie charts
Spending VisualizationCategory breakdown charts of actual spending
Budget HistoryMonth-by-month budget vs. actual comparison with trend charts
Predictive Cash Flow6-month savings forecast based on current spending rate
AI Chat AssistantRule-based financial Q&A assistant using live budget data
User ProfilePhoto upload, password change, account deletion
